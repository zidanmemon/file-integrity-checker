# File Integrity Checker

A simple Python tool that generates the SHA-256 hash of any file.

## Features

- Generate SHA-256 hash
- Supports any file type
- Automatic quote removal from file path
- Error handling for invalid file paths

## Technologies Used

- Python
- hashlib
